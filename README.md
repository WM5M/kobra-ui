# KOBRA UI

Red/Black NUI menu for FiveM cheat UI integration.

Files in repo root:

- `index.html`
- `style.css`
- `ui.js`

After first commit, get a Githack CDN URL in this form:

`https://rawcdn.githack.com/WM5M/kobra-ui/<COMMIT_HASH>/index.html`

Use that URL in your Lua:

```lua
DUI = MachoCreateDui("https://rawcdn.githack.com/WM5M/kobra-ui/<COMMIT_HASH>/index.html")
```
